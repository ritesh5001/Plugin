Creates SLOWY content pages except the homepage.

== Description ==

This plugin creates and updates the following pages:

* About Us
* Collections
* Contact Us
* Privacy Policy
* Terms & Conditions
* Return & Refund Policy
* Cancellation Policy
* Shipping Policy

If WooCommerce is active, it also creates the SLOWY product category structure.

== Installation ==

1. Upload the `slowy-pages` folder to `/wp-content/plugins/`.
2. Activate `SLOWY Pages` from the WordPress admin.
3. Review generated pages and add them to menus as needed.

== Notes ==

The plugin does not create or set a homepage.
Generated pages are updated when the plugin content version changes.
