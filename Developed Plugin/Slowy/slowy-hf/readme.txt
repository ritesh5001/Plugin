Auto-injects SLOWY branded header and footer site-wide.

== Description ==

This plugin adds a SLOWY header and footer with:

* Logo image from http://slowy.in/wp-content/uploads/2026/06/Slowy.png
* Editable logo URL
* Editable navigation, account, cart, search, social, and policy links
* Editable email, phone, WhatsApp, address, and support hours
* WooCommerce-aware shop, account, cart, search, and cart count defaults
* Shortcodes: [slowy_header] and [slowy_footer]

Settings are available under Appearance > SLOWY Header & Footer.

== Installation ==

1. Upload the `slowy-hf` folder to `/wp-content/plugins/`.
2. Activate `SLOWY Header & Footer` from the WordPress admin.
3. Go to Appearance > SLOWY Header & Footer to change links and contact details.
